# PowerShell 配置文件

这是一个简洁的 PowerShell 配置文件，包含以下特性：

- Oh My Posh 主题
- PSReadLine 智能预测和快捷键设置
- Terminal-Icons 支持
- Winget 自动补全
- 实用函数和别名

## 安装

1. 克隆此仓库到您的 PowerShell 配置文件目录
2. 在 PowerShell 中运行 `. $PROFILE` 以加载配置

## 使用

配置文件会自动加载。享受增强的 PowerShell 体验！
