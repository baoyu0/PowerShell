# PowerShell 配置文件

这是一个功能丰富的 PowerShell 配置文件，包含以下特性：

- 自动代理设置
- 常用工具安装和更新
- 自定义提示符
- 插件系统
- 国际化支持
- 单元测试

## 安装

1. 克隆此仓库到您的 PowerShell 配置文件目录
2. 在 PowerShell 中运行 `. $PROFILE` 以加载配置

## 使用

- 使用 `Show-ProfileMenu` 或别名 `s` 打开配置文件管理菜单
- 使用 `Update-Profile` 手动更新配置文件
